## Project Overview
Analysis of 110,527 medical appointments to identify factors driving no-shows
and develop targeted intervention strategies.

## Key Findings
- **No-show rate:** 20.2% (costing $13.4M annually)
- **Top factors:** Lead time, patient history, age, neighborhood
- **Opportunity:** $2.65M annual savings identified
